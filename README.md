# 家谱系统

这是一个基于 Tkinter 的家谱可视化程序，当前版本采用纯 Canvas 图谱展示，界面风格接近“穿透图”。

## 当前功能

1. 纯可视化展示家谱关系，父母在上、本人居中、子女在下、兄妹同层。
2. 启动时自动生成 10 代以上的测试数据，便于直接验证界面效果。
3. 支持鼠标滚轮缩放、左键拖拽平移图谱。
4. 鼠标悬停节点时显示人物气泡详情。
5. 右键节点可执行设为中心、编辑、删除、添加父母、添加子女。
6. 数据自动保存到项目根目录的 data.json。

## 项目文件

1. [main.py](main.py)：程序入口与全部 UI / 数据逻辑。
2. [README.md](README.md)：项目说明。
3. `data.json`：运行后自动生成的数据文件。

## 数据模型

家谱基础元素是“人”，包含以下字段：

1. 姓名
2. 性别
3. 生日
4. 卒日
5. 父亲
6. 母亲
7. 生平简介

兄妹和子女不单独存储为数组，而是通过父母关系动态推导，避免关系重复维护。

## 运行方式

```bash
cd c:\Users\vipis\jiapu-system

```

如果你希望重新生成内置样例数据，可以先删除 `data.json` 再启动程序。

## 交互说明

1. 鼠标悬停节点：查看气泡详情。
2. 鼠标右键节点：打开操作菜单。
3. 鼠标滚轮：缩放图谱。
4. 按住左键拖拽：平移图谱。
5. 点击“重置中心”：回到根人物视角。
